# Furniture-Mall-System
基于 Spring Boot + Vue 的家具商城系统，是一个集家具销售、展示的平台。系统具有家具分类、搜索、详情查看、下单、支付、订单管理等功能。前端 Vue 提供美观便捷的界面，方便用户操作；后端 Spring Boot 确保系统稳定，保障商品信息和用户数据的安全存储与高效处理，为用户提供优质的家具购物体验，助力商家拓展业务，推动家具销售。
